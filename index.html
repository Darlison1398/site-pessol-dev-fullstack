Teste automatizado

Nesse projeto, eu mostro um teste que foi construído usando JUnit com Selenium, em Java. Eu construí o teste para acessar o website Clima Temp e realizar várias operações que uma pessoa comum faria. O teste verifica se o nome da cidade escolhida é mostrado corretamente, se o tempo está apresentado em graus Celsius e se os botões com as sugestões de cidades estão funcionando corretamente.

CrudApp

Essa aplicação foi desenvolvida em ASP.NET Core, Bootstrap e SQL Lite, e foi hospedada na Microsoft Azure. Nela, eu usei uma arquitetura MVC para criar um sistema de registro de cards. Um dos destaques dessa aplicação é a implementação de um pipeline de CI/CD (Integração Contínua e Entrega Contínua). Configurei o pipeline no GitHub Actions para automatizar o build, os testes e o deploy da aplicação na nuvem, garantindo um fluxo ágil e eficiente de atualizações.
