Asp Gestor

Esta aplicação foi criada para ajudar profissionais autônomos a organizar seus ganhos e despesas mensais de forma prática e segura. Com ela, é possível acompanhar facilmente o lucro diário, mensal e líquido, além de visualizar os dados em painéis interativos.

Principais funcionalidades:
- Registrar ganhos;
- Registrar gastos/despesas;
- Cálculos automáticos dos ganhos e gastos;
- Informações sobre o lucro diário, lucro mensal e lucro líquido;
- Dashboards animados que facilitam na visualização dos dados;
Processo de desenvolvimento até o deploy:
- O Front-end dela foi desenvolvida em Angular e Bootstrap;
- Back-end foi desenvolvido em Asp.Net Core;
- O banco de dados usado iniclamente foi o SQL Server, mas depois eu migrei para o PostgreSQL;
- JWT para autenticação;
- Git e Github para versionamento;
- Foi hospedada na Azure, mas migrei para VPS;
- Implementei um pipeline de Integração e Entrega Contínua (CI/CD) utilizando GitHub Actions, automatizando as etapas de build, execução de testes e deploy da aplicação em uma VPS

CrudApp

Essa aplicação foi desenvolvida em ASP.NET Core e Bootstrap. O projeto foi hospedado em um servidor VPS e utiliza o banco de dados SQLite. Nela, eu usei uma arquitetura MVC para criar um sistema de registro básico de cards. Um dos destaques dessa aplicação é a implementação de um pipeline de CI/CD (Integração Contínua e Entrega Contínua). Configurei o pipeline no GitHub Actions para automatizar o build, os testes e o deploy da aplicação na VPS, garantindo um fluxo ágil e eficiente de atualizações.
